# Assets MSConecta

Coloque aqui os arquivos de arte-final e depois **copie-os para `../public/`**
para que o Remotion consiga carregá-los via `staticFile()`.

| Arquivo              | Descrição                                   | Destino                      |
|----------------------|---------------------------------------------|------------------------------|
| logo-triangles.png   | Ícone triangular azul (canto sup. direito)  | `public/logo-triangles.png`  |
| logo-white.png       | Logotipo branco sem fundo (bloco de título) | `public/logo-white.png`      |
| watermark.png        | Versão ultra-transparente para overlay      | `public/watermark.png`       |

## Fonte
Coloque `BarlowCondensed-Bold.ttf` em `public/fonts/`.
